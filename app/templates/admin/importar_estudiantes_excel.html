Subir Archivo de Excel (.xlsx)
Requisitos del Archivo

El sistema procesará el archivo de Excel esperando la siguiente estructura de columnas:

  • Columna A: Sección (Debe coincidir con un Departamento existente)
  • Columna B: Estudiante (Formato: Apellidos, Nombres)
  • Columna C: Grupos (Contendrá 'ALMUERZO NORMAL' o 'ESPECIAL')
  • Columna D: Id (El código del estudiante)
  • Columna E: Sexo
Nota: El sistema actualizará los estudiantes existentes basándose en su 'Id'. Si un 'Id' del archivo no se encuentra en la base de datos, se creará un nuevo estudiante.
